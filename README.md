# Operato Lighting 설치 및 셋업 문서

## 설치 : 

아래 명령어 실행 시 자동으로 http://127.0.0.1:4100/ 로 서버가 실행 됨

```sh
#=> 지킬 설치
$ gem install bundler jekyll
#=> 리파지토리로 이동
$ cd operato-lighting/docs
#=> local dependency설치
$ gem install bundler 
#=> 서버 실행
$ bundle exec jekyll serve
```
LiveReload Server를 사용하기에 편집 시 자동으로 브라우저가 리프레시됨.
